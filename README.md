# DNShell         
一款基于DNS C2隧道的反弹shell工具。         
         
## 支持         
**功能：**         
支持DNS-recordA-直连型 的C2隧道。         
         
**目标：**         
Windows下基于Powershell的反弹。         
Linux下基于ShellScript的反弹。
